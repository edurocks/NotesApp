# NotesApp
Application to showcase an app that you can insert, delete, update your notes. The backend was built with <b>Ktor</b>. It works online and offline.

# Features
- MVVM 
- RecyclerView
- SwipeToDelete
- Syncronization System
- Database Caching
- Dagger Hilt
- Text Markdown
- Dialogs
- Encrypted Shared Preferences
- Login System
